Application Walkthrough

  1. Take a Picture or Upload:

    To get started, you have two options: either take a new picture of your outfit using the "Snap" followed by "Save" button, or upload an existing image from your device using the "Upload" button.

  2. Style your Top:

    After the image is loaded, click on "Style" to describe the top in the image.

    A description will be generated in the text box below.

  3. Visualize Recommendations:

    Click on "Show" to see image recommendations for style, based on your input.

    Each image recommendation has a description at the back - click on the image to flip and view that.

  4. View Generated Outfit:

    The application will also try to generate an outfit using the image you have input. Please find that generated outfit in the bottom of the page.

If you have any further questions, please feel free to contact support.
